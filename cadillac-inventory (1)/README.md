# 🏎️ CADI — Cadillac Inventory Management System

A premium, single-page Cadillac vehicle inventory management web app with a dark luxury aesthetic.

## ✨ Features

- **Browse Inventory** — Grid and list views with smooth transitions
- **Add Vehicles** — Full form with model, trim, year, price, mileage, VIN, colors, and more
- **Edit & Delete** — Inline editing with confirmation dialogs
- **Image Gallery** — Upload multiple photos per vehicle with a gallery viewer
- **Search & Filter** — Real-time search, filter by status, model, and sort by price/year
- **Persistent Storage** — All data saved to `localStorage` (no backend needed)
- **Responsive** — Works on desktop and mobile

## 🎨 Design

- Dark luxury theme — black, gold, and chrome
- Cormorant Garamond + Outfit typography
- Glassmorphism accents and micro-animations
- Cinematic hero section

## 🚀 Getting Started

No build step required. Simply open `index.html` in any modern browser.

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/cadillac-inventory.git

# Open in browser
open index.html
```

## 📋 Cadillac Models Supported

- Escalade / Escalade ESV
- CT4 / CT4-V / CT4-V Blackwing
- CT5 / CT5-V / CT5-V Blackwing
- LYRIQ / OPTIQ / VISTIQ
- XT4 / XT5 / XT6

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — No dependencies, no frameworks
- **localStorage** — Client-side data persistence
